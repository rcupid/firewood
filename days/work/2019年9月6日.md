## 本周遇到的问题 总结 & 思考
Vue的jsx绑定双击事件，写法和react有很大的区别。React的写法，更接近原生写法   
### Vue的写法
```javascript
render(){
  return (
    // 都是小写
    <button ondblclick={e => { console.log(e) }}></button>
  )
}
```

### React的写法
```javascript
render(){
  return (
    <button onDoubleClick={e => { console.log(e) }}></button>
  )
}
```
