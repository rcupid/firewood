### [Vim命令记不住？可能是姿势不太对[视频]](https://zhuanlan.zhihu.com/p/66166188)
- i -->insert 
- a -->append
- o --> open line
- c -->change
 -- c+i+w 
 -- c+t+位置
- 撤销 u  --->undo
- 单词间移动
 - w word
 - b back word
- 行间移动
 - ^ 行首
 - $ 行尾
 - 0 行首
 - gg 第一行
 - G  最后一行
 - #G  例如：3G  跳转到第3行
 - :行号  例如：  ：3 跳转到第3行

- 翻页
 - ctr+f (forward)
 - ctr_u (upward) 
- 查找单词
  - /word 向后查找
  - ?word 向前查找
  - f+字符 在某一行内，跳转到字符开头的单词
